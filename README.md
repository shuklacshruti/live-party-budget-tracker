# live-party-budget-tracker
Live party budget tracker with customizable quotes

Live editing:

Click any item name to rename it inline
Type directly into Est Low / Est High fields as quotes come in
Fill in the Actual $ column once you confirm a vendor, the tracker auto-calculates over/under budget
Edit category names right in the header

Tracking:

The top cards show Total Budget, Actual Spent, Remaining, and Estimated Range in real time
The progress bar fills up as you log actual costs (turns red if you go over)
Set your own budget cap (default $2,000) and it recalculates instantly
Each category badge shows either estimated range or actual spend

Customizing:

+ Add Item inside any category to add new line items
+ Add Category button (or header button) to add new sections with a custom emoji
🗑 icon to delete any category or row
⬇ Export CSV to save a spreadsheet copy
